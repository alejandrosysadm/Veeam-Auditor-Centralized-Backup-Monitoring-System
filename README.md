# Veeam-Auditor-Centralized-Backup-Monitoring-System
Veeam Auditor es una solución de automatización diseñada para centralizar, monitorizar y auditar reportes de backup provenientes de Veeam Backup &amp; Replication. El sistema transforma una bandeja de entrada de Outlook saturada en una base de datos estructurada y un dashboard interactivo de alto rendimiento.
